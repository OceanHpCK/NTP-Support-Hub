Để tài liệu PDF của bạn hiển thị đúng trên website, bạn cần copy file vào đúng thư mục và đặt tên file chính xác như bảng bên dưới.

## Lưu ý quan trọng:
1.  **Định dạng:** Tài liệu bắt buộc phải là file `.pdf`.
2.  **Tên file:** Phải viết thường, không dấu, không khoảng trắng (như cột "Tên file cần đặt").
3.  **Cập nhật:** Khi bạn muốn thay đổi nội dung một tài liệu, chỉ cần copy đè file mới có cùng tên vào đúng thư mục. Web sẽ tự động nhận diện nội dung mới mà không cần khởi động lại hệ thống.
4.  **Kiểm tra:** Nếu file không hiển thị, hãy kiểm tra xem tên file trong folder có khớp 100% với tên file ghi chú trên trang Web (phần *File: xxxx.pdf*) hay không.
